十独特功能

opcode GitHub
  • Checkpoint 系统 — Git 式快照，支持手动/自动（smart/per-prompt/per-tool），可 fork 分支
  • Usage Dashboard — 按模型/日期/项目统计 token 和费用
  • Web 模式 — Axum 服务器 + WebSocket，手机远程访问
  • GitHub Agent 浏览器 — 从 GitHub 导入社区 Agent
Any-code GitHub
  • Auto-compact 上下文 — 智能 token 监控 + 自动上下文压缩
  • 智能翻译中间件 — 8 种内容提取策略，后台渐进式翻译
  • 消息回滚 — 撤回 prompt 并自动回退到执行前状态
  • 费用追踪面板 — 多模型定价、cache hit 率分析
codexia GitHub
  • P2P 远程控制 — QUIC 隧道（桌面服务端 + iOS 客户端）
  • Cloudflare Tunnel — 远程访问无需暴露端口
  • 自动化调度 — Cron 定时任务，带执行历史
  • 插件/技能系统 — 可扩展 slash 命令
  • 笔记系统 — 内置笔记本，SQLite 持久化
  • 分析面板 — 贡献热力图、token 分解
metabot GitHub
  • Agent Bus — Bot 间通讯（mb talk <bot>）
  • MetaSkill — 一键生成多 Agent 团队
  • Peer Federation — 跨实例 Bot 发现与路由
  • MetaMemory — 嵌入式知识库 + 全文搜索
  • 火山 RTC 语音 — ASR→LLM→TTS 全链路 + 字幕
  • 团队预算管理 — 每日预算 + Prometheus + 熔断器
  • 飞书技能集 — 19 个预集成 AI 技能
happy GitHub
  • 零知识加密 — E2E 加密（TweetNaCl + AES-256-GCM）
  • fd3 遥测管道 — 自定义 fd 拦截 API 状态
  • 多设备同步 — 桌面/手机/Web 实时同步
  • 推送通知 — 权限/完成等推送到手机
  • Daemon 守护进程 — 后台常驻，版本感知
  • 设备配对 — 加密密钥交换
cc-connect GitHub
  • 7 种 Agent 适配器 — Claude、Codex、Cursor、Gemini、Qoder、OpenCode、iFlow
  • 10 种 IM 平台 — 飞书、钉钉、Telegram、Slack、Discord 等
  • Provider Proxy — 本地反代改写第三方 thinking 参数
  • 6 种权限模式 — 最完整的权限控制
  • 自然语言 Cron — 聊天中设定定时任务
  • 附件回传 — Agent 文件自动推送到聊天
  • 多语言 i18n — 中/英/日/西/繁体
nexu GitHub
  • 双击安装 — 全图形化设置，零 CLI 配置
  • OpenClaw 控制平面 — 本地管理 agent 生命周期
  • 10+ IM 渠道 — 开箱即用的多平台连接
  • 多模型 BYOK — 10+ 提供商一键切换
  • OAuth 登录 — MiniMax、GLM 免密登录
  • Skill 热重载 — 技能目录实时监听
  • 本地优先 — 数据全部存本地，无订阅
